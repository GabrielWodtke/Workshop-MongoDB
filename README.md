

# Sobre o projeto

O projeto foi criado junto com o Workshop do professor Nélio alves com o objetivo de gerenciar usuários, posts e comentarios de uma rede social usando MongoDB e Spring Boot como ferraementas principais.

## Modelo conceitual
![Modelo Conceitual](https://github.com/GabrielWodtke/Workshop-MongoDB/blob/main/assets/Screenshot_1.png)

# Tecnologias utilizadas
## Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven
- MongoDB


# Como executar o projeto

## Back end
Pré-requisitos: Java 24, Spring Boot 4

## Implantação em Produção

Banco de dados: MongoDB

```bash
# clonar repositório
git clone https://github.com/GabrielWodtke/workshop-springboot-jpa.git
# entrar na pasta do projeto back end
cd backend

# executar o projeto
./mvnw spring-boot:run
```
